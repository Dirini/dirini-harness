# Writing System

이 디렉토리는 글쓰기 자산을 `형식(format)`과 `목적(purpose)`으로 나눠 관리하기 위한 구조 예시다.

핵심 원칙:

- 형식은 글의 겉모양과 기본 구조를 결정한다
- 목적은 평가 기준과 논리 방식을 결정한다
- 에이전트는 공통 파이프라인으로 두고
- 형식/목적 폴더는 참고 자산과 루브릭을 저장하는 용도로 쓴다

## 구성

- `formats/`: 독후감, 에세이, 보고서, 성찰문, 자기소개 등
- `purposes/`: 분석, 설득, 성찰, 지원, 정보전달 등
- `rubrics/`: 과목별, 교수별, 공통 평가 기준
- `samples/`: 직접 쓴 샘플 글
- `voice-rubrics/`: 샘플에서 뽑은 말투 규칙
