# Assignment Brainstorming

과제형 글쓰기를 시작하기 전에 먼저 써야 할 것을 분명하게 만든다.

## 목표

- 무슨 글을 써야 하는지
- 누구를 향해 쓰는지
- 어떤 주장과 구조를 가져갈지
- 어떤 자료가 필요한지

를 초안 작성 전에 명확하게 정리한다.

## 체크할 것

1. 과제 종류
   - 독후감 / 서평 / 에세이 / 보고서 / 발표문 / 반성문 / 기타
2. 평가자
   - 교수 / 조교 / 심사자 / 팀원 / 일반 독자
3. 목표
   - 감상 전달 / 분석 / 비교 / 주장 / 설명
4. 제약
   - 글자 수, 형식, 인용 규칙, 마감
5. 자료
   - 책, 기사, 수업노트, 본인 메모, 직접 쓴 샘플

## 출력

```markdown
## Assignment Brief
- 과제 종류:
- 독자:
- 목표:
- 핵심 주장 또는 질문:
- 필요한 근거:
- 형식 제약:
- 먼저 확인할 자료:
```

## 원칙

- 초안보다 브리프가 먼저다
- 애매한 목표는 반드시 한 문장으로 줄인다
- "좋게 쓰기"는 목표가 아니다
- 무엇을 보여줘야 통과인지부터 정리한다
