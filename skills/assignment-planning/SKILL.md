# Assignment Planning

과제형 글쓰기의 브리프를 실제 작성 가능한 목차와 논리 흐름으로 바꾼다.

## 언제 쓰나

- Assignment Brief가 나온 뒤
- 초안을 바로 쓰면 방향이 흔들릴 것 같을 때
- 독후감이나 보고서를 문단 구조까지 미리 설계하고 싶을 때

## 계획 방식

1. 중심 질문 1개
2. 핵심 주장 1-3개
3. 각 주장에 붙는 근거
4. 문단별 역할
5. 결론에서 남길 인상

## 출력

```markdown
## Writing Plan

### Thesis
- ...

### Section Outline
1. 도입:
2. 본문 1:
3. 본문 2:
4. 본문 3:
5. 결론:

### Evidence Map
- ...

### Writing Risks
- ...
```

## 원칙

- 문단마다 역할이 달라야 한다
- 근거 없는 문단은 만들지 않는다
- 감상문이어도 구조는 필요하다
- 초안은 이 계획을 구현하는 단계다
