# Style Learning

사용자 말투를 샘플에서 추출해 재사용 가능한 기준으로 만든다.

## 언제 쓰나

- "내 말투로 써줘"가 중요한 작업일 때
- 사용자가 과거에 직접 쓴 파일을 줄 수 있을 때
- 과제, 독후감, 자기소개, 에세이에서 말투 일관성이 중요할 때

## 절차

1. 샘플 요청
   - 직접 쓴 글 2-5개
2. 샘플 분해
   - 문장 길이
   - 어조
   - 논리 전개
   - 자주 쓰는 표현
   - 피해야 할 표현
3. Voice Rubric 작성
4. 초안에 적용
5. 초안이 Rubric을 지켰는지 다시 점검

## 출력

```markdown
## Style Learning Output

### Voice Rubric
- ...

### Keep
- 유지할 특징

### Avoid
- 피해야 할 특징

### Drafting Notes
- 초안 쓸 때 유의점
```

## 원칙

- 말투는 분위기가 아니라 규칙이다
- "비슷하게"가 아니라 무엇이 비슷해야 하는지 적는다
- 샘플보다 더 멋있게 쓰려 하지 말고, 샘플과 합치되게 쓴다
