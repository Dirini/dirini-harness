## 조민 (Jomin)

AI Native. 학생 + 창업/사업 준비 중. Claude Code로 사고 과정과 글쓰기를 자동화하고, 그 시간으로 더 큰 가치를 만든다.

### 핵심 사고 원칙 10가지

1. **주체성 (Agency First)** — 문제를 해결할 사람은 나. 기다리지 않고 직접 움직인다
2. **암묵지 → 명시지** — 무의식적 판단을 말/숫자로 꺼내야 자동화 가능. "괜찮은 것 같은데" → "72점"
3. **Unknown Unknowns 연쇄** — 하나를 풀면 세 개의 모르는 게 나온다. 이 연쇄를 즐긴다
4. **복리 성장 (Flywheel)** — 암묵지→명시화→발견→루프→더 깊은 명시화. 지수 성장
5. **판단의 해상도** — 감이 아니라 숫자로. 세계가 선명해진다
6. **만드는 비용 0 → 판단력이 핵심** — 도메인 전문성 + 무엇을 만들지 고르는 눈
7. **도구가 아니라 구조화** — "Claude Code 이렇게 쓰세요"가 아니라 "당신의 판단 기준이 뭐예요?"
8. **시스템 사고** — 정답 찾기 → 학습하는 구조 만들기
9. **실행하게 만드는 정보** — 행동 = 동기 + 능력(쉬움) + 트리거 (Fogg Model)
10. **Engineering as Marketing** — 만든 도구가 곧 리드 마그넷
